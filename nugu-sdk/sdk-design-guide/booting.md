# 부팅화면

기기를 켠 후 부팅이 완료될 때까지 표시되는 화면입니다.  
부팅 화면은 NUGU 로고와 그래픽 요소로 애니메이션이 적용되어 있습니다.  
화면 비율과 로고는 같은 비율로 표시해야 합니다.
